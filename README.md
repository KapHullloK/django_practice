## О проекте
Учебный проект на Django и Django REST Framework для управления видеоконтентом через HTTP API.

## Какую задачу решает
Проект реализует базовый сервис видео: хранение роликов и их файлов в разных качествах, контроль доступа к контенту, учет лайков и получение статистики по авторам.

## Основное назначение
Репозиторий предназначен для практики проектирования REST API, работы с кастомной моделью пользователя, разграничения прав доступа и построения запросов к PostgreSQL.

## Ключевой функционал
- API списка и детальной карточки видео (`/v1/videos/`).
- Доступ к опубликованным видео для всех и к приватным видео для владельца/администратора.
- Лайки для опубликованных видео с защитой от повторного лайка.
- Хранение нескольких файлов одного видео (HD/FHD/UHD).
- Админские эндпоинты для списка ID опубликованных видео и статистики лайков по пользователям (через `Subquery` и `GROUP BY`).
